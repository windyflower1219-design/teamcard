# PDF 파싱 문제 해결 방안

## 문제
- PDF에서 단락이 제대로 분리되지 않음
- 전체 텍스트가 하나로 합쳐져서 표시됨

## 원인
PDF.js가 텍스트를 추출할 때 줄바꿈이 공백으로 변환되어 정규식 패턴이 작동하지 않음

## 해결 방법
1. PDF 텍스트 추출 시 줄바꿈 정보 보존
2. 단락 번호 패턴 개선 (1-1. 형식)
3. 최대 문자 수 제한으로 단락 분리

## 테스트 정보
- 파일: ARN6672_ATP 3-21x20 FINAL WEB.pdf
- 파싱 페이지: 13-41
- Glossary 페이지: 547-573
